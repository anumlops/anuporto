## Overview

Built a routing system that analyzes prompt complexity and dispatches to the most cost-effective model.

## Architecture

Uses a lightweight classifier to detect task type (code, reasoning, creative) and routes accordingly.

## Results

42% cost reduction on API bills while maintaining response quality.
